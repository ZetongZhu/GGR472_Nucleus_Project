Mapping Food Access for Nucleus in Halton Region

An Interactive Tool for Service Planning and Spatial Insight

What We Create for U!

Our interactive map provides Nucleus Independent Living with actionable geographic insights by:

  • Identifying Underserved Neighborhoods: Dissemination Areas are color-coded based on proximity to food banks, highlighting neighborhoods with limited food bank accessibility.
  • Evaluating Transit Accessibility: Visualizes the number of bus stops within a 1,000-meter radius around food banks, enabling assessment of public transit access to food assistance.
  • Detailed Household Analysis: Address points are mapped with proximity to bus stops, facilitating tailored analyses for client-specific transit accessibility.

This systematic spatial analysis empowers Nucleus to strategically enhance their food accessibility initiatives and target interventions effectively.

Methodology

We conducted spatial analysis through a combination of ArcGIS Pro and ArcGIS Online.

  • Data Processing (ArcGIS Pro): We used spatial query, Near function, and Buffer tools to calculate proximity and service areas.
  • Layer Symbolization: Layers were styled using color gradients and spatial buffers to reflect accessibility.
  • Web Deployment: Processed data was uploaded to ArcGIS Online, with pop-ups and interactive legends configured for user experience.
  • Interface Design: The website was developed using HTML + Bootstrap to provide a responsive, accessible layout.

These steps helped deliver a scalable decision-support tool that visually communicates where food access gaps exist in the Halton Region.

How to Use the Map

Based on the type of geospatial relationship you intend to explore, select the correct layer that is relevant to your exploration. This map was designed to reflect three key types of spatial relationships that provide implications for Nucleus to review and plan for their services:

  1. Dissemination Area (DA) and Food Banks: Color-gradient proximity maps help identify DAs that are underserved by food bank coverage.
  2. Food Banks and Public Transit: Buffer-based analysis (1KM radius) highlights how well each food bank is served by nearby bus stops.
  3. Residential Address Points and Bus Stops: Proximity-based relationships illustrate transit accessibility for specific household locations. Especially useful for query based on clients' address.

This map contains multiple layers. Here's how to interpret them:

  • Halton Food Banks: Click black icons to view name, hours, and contact.
  • Spatial Join Buffers: Purple rings created by 1KM Buffer show how many bus stops are near each food bank.
  • Dissemination Areas (DA): Colored polygons showing the distance between food banks for each area centre.
  • Address Points: Home locations used for possible client-specific accessibility analysis.
  • Bus Stops: Bus signs representing public transit access.
